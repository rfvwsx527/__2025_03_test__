# Git 基礎教學

## 1. Git 簡介
- Git 是什麼
- 為什麼要使用版本控制
- Git 的優勢

## 2. 基本設定
- 安裝 Git
- 設定使用者資訊
- 創建 Git 倉庫 (Repository)

## 3. 基礎指令
- git init - 初始化倉庫
- git add - 加入暫存區
- git commit - 提交變更
- git status - 查看狀態
- git log - 查看歷史紀錄

## 4. 分支管理
- 什麼是分支
- 建立分支
- 切換分支
- 合併分支
- 解決衝突

## 5. 遠端操作
- 什麼是 GitHub/GitLab
- git remote - 管理遠端倉庫
- git push - 推送到遠端
- git pull - 從遠端拉取
- git clone - 複製遠端倉庫

## 6. 進階技巧
- git rebase - 重整歷史
- git stash - 暫存修改
- git tag - 版本標籤
- .gitignore - 忽略檔案
